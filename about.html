Questions:

  1. What does the server in the client architecture do?
    Responds to requests made by the user. The server transmits and recieves files to the client.
  2. What does the client do?
    The client, once connected to a browser, can locate and display information from the web.
  3. What is an IP adress?
    IP stands for Internet Protocol, and is an unique identifier for each computer to compute over a network.
  4. What is DNS?
    DNS stands for Domain Name System, and is the "phone book" for IPAdresses, so you can use names for webpages instead of IPAdresses
  5. What is the difference between the Internet and WWW?
    The internet is a global connection between computers. WWW is a is a collection of information which is accessed via the Internet. The internet is the infrastrurcture, while WWW is a service on top.
  6. Explain what HTML is?
    HTML is the standard markup language for web pages. It stands for The HyperText Markup Language. It what we use to code this page, and for the browser to load what we want to.
